
To run or modify this application

1. Install latest Node.js (Windows or Linux) 

2. Install NPM for the Node.js command line inside the command line
    - "npm -v"
    - "npm install npm@latest -g"

3. Install Vue.js CLI on the command line
    - "npm install -g @vue/cli"

4. Install yarn on the command line
    - "npm install -g yarn"

4. Move to the vspp folder 
    - "cd ..."

5. Install telephone validation library 
    - "yarn add vue-tel-input"
    - "npm i --save vue-tel-input"

6. Serve Vue.js application
    - "yarn serve"
    - Command line console log should be like the following
    "App running at:
    - Local:   http://localhost:8080/
    - Network: http://192.xxx.xxx.x:8080/

    Note that the development build is not optimized.
    To create a production build, run npm run build."

7. Open up a browser at http://localhost:8080/

8. Install latest Visual Studio Code

9. Install latest Vetur on top of Visual Studio Code for Vue.js development
